> AI + ML + Live Telemetry Systems for sim racing performance optimization

## Overview

SC Tuning is a personal project exploring AI, ML, and live telemetry systems through the lens of sim racing performance optimization.

**Two focus areas:**

- **AI + ML + Telemetry Systems** - Real-time pipelines, adaptive agents, and ML systems for telemetry processing and performance insights
- **Forza Applied Tuning Lab** - Applying these tools to improve driving and tuning in Forza Motorsport

## Development

```bash
# Install dependencies
npm install

# Run local server
hugo server

# Build for production
hugo
```
